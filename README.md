# fa25sbomproject
Fall '25 Faculty SBOM Project

The goal of the project is to learn how SBOMs function and create a user-friendly SBOM CLI application that mainly provide generate and analyze capabilities.

The SBOM CLI application satisfies the NTIA Minimum Elements for an SBOM while being SPDX 2.3 compliant.
